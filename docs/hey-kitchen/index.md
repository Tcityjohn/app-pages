# Hey Kitchen

Your voice-powered kitchen assistant for unit conversions, recipe scaling, and cooking timers.

## Links

- [Privacy Policy](./privacy-policy)
- [Support](./support)

## Features

- **Voice Commands** - Convert units, scale recipes, and set timers hands-free
- **Recipe Scaling** - Double, triple, or halve any recipe instantly
- **Unit Conversion** - Cups to grams, tablespoons to ml, and more
- **Kitchen Timers** - Set multiple named timers while you cook
